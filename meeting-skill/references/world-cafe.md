# 世界咖啡 (World Café)

## 方法概述

通过小组轮换、桌长记录、多轮对话的方式，让不同成员围绕核心问题交叉讨论。

## 核心机制

| 元素 | 说明 | Agent适配 |
|------|------|-----------|
| 小组 | 固定小组讨论 | 分组讨论 |
| 桌长 | 记录关键观点 | 每组指定记录者 |
| 轮换 | 小组轮换交流 | 轮换讨论话题 |
| 汇总 | 回收成果 | TeamLead汇总 |

## Agent适配流程

### Phase 1 - 设定核心问题

**TeamLead广播**：
```
【世界咖啡】核心问题
问题1: {问题}
问题2: {问题}
问题3: {问题}
轮次: {N}轮
每组人数: {N}人
```

### Phase 2 - 分组讨论

**Round 1**：
```
【世界咖啡 Round 1】
问题: {问题1}
各组讨论{分钟}
桌长负责记录关键观点
```

**小组输出模板**：
```markdown
## 小组{N} - Round 1 记录

### 问题：{问题1}

### 关键观点
1. ...
2. ...

### 待探讨问题
- ...

### 桌长：{Agent名称}
```

### Phase 3 - 轮换讨论

**Round 2/3**：
```
【世界咖啡 Round {N}】
问题: {问题N}
各组讨论，可参考上轮其他组的记录
桌长记录新观点
```

**小组输出模板**：
```markdown
## 小组{N} - Round {N} 记录

### 问题：{问题N}

### 本组观点
1. ...

### 参考其他组的延展
- 延展小组{M}的观点{N}：{延展内容}

### 待探讨问题
- ...
```

### Phase 4 - 全体汇总

**TeamLead汇总**：
```markdown
## 世界咖啡汇总

### 高频观点（多组提及）
| 观点 | 提及组数 | 具体内容 |
|------|----------|----------|
| ... | {N} | ... |

### 共识方向
- ...

### 关键分歧
- ...

### 待深入问题
- ...
```

## 适用场景

- 参与人数较多
- 议题较开放
- 需广泛吸收观点
- 跨部门知识分享

## 注意事项

- 收敛较慢，不适合快速决策
- 知识和视角在不同小组流动
- 桌长记录很重要
- 最后需汇总出共识方向
