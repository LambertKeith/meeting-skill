# 会议案例库说明

## 概述

本案例库累积各类会议的完整案例，供新会议参考议程配置和方法组合。

---

## 案例库用途

1. **参考议程配置**: 新会议可参考类似案例的议程结构
2. **方法组合选择**: 了解不同场景的方法组合效果
3. **输出样例参考**: 查看各类会议的标准输出格式
4. **时间估算参考**: 根据案例预估会议时长

---

## 案例结构标准

每个案例包含：

### 1. 案例信息

| 项目 | 内容 |
|------|------|
| 案例编号 | case-{type}-{seq} |
| 会议类型 | {类型} |
| 使用方法 | {方法组合} |
| 参会Agent数 | {N} |
| 环节数 | {N} |
| 总时长 | {分钟} |
| 效果评分 | {1-5} |

### 2. 用户触发

用户原始请求，用于匹配类似场景。

### 3. 预期流程

按阶段描述会议流程：
- Phase 1: 会议启动
- Phase 2-N: 环节执行
- Phase N+1: 会议收尾

### 4. 完整输出示例

包含所有输出文件的内容示例：
- agenda.md
- phases/*.md
- minutes.md
- action-items.md

## 案例使用方式

### 1. 按场景查找

根据会议类型和议题特征，在对应目录查找类似案例。

### 2. 参考议程配置

查看案例中的agenda.md，参考：
- 环节划分
- 时间盒设置
- 方法组合

### 3. 参考输出格式

查看案例中的输出文件，了解标准格式。

### 4. 调整适配

根据实际需求调整，不照搬案例。

---

## 案例贡献指南

### 新案例添加时机

- 完成一次效果良好的会议
- 使用了新的方法组合
- 解决了特殊场景问题

### 案例撰写要求

1. **完整性**: 包含所有必需输出文件
2. **真实性**: 基于真实会议整理
3. **可复用**: 他人在类似场景可参考
4. **有评价**: 包含会议效果评价

### 案例文件命名

```
case-{seq}-{topic}.md

示例：
case-001-project-delay.md
case-002-bug-analysis.md
```
