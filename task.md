# 任务

本项目的目标是构建一个用于组织多agent会议的agent skill


# 方法

需要使用skill-creator进行编写，符合原则

需要参考 https://github.com/tmstack/awesome-persona-skills 中汇总的多个skill项目的编排和撰写方法，维持风格一致性


# 开会的方法论

文件位于 @开会的方法论.md

你需要参考，并将其适当的改造，因为我们不是让人去开会，而是让agent去开会

在这些方法之外，还需要有会议纪要的撰写方法，以及会议记录的撰写方法，原则上要让每个角色在每个环节的输出内容都要有所记录，然后整体流程也需要有所记录
