# 脑写法 (Brainwriting)

## 方法概述

比口头头脑风暴更稳定的方式。先独立写下想法，再交换、补充、延展。

## 常见变体：6-3-5法

- 6个人
- 每人写3个想法
- 传递5轮

## Agent适配流程

### Phase 1 - 独立书写

**TeamLead广播**：
```
【脑写法 - Round 1】独立书写
任务：每人独立写下3个想法
规则：不参考他人，纯粹独立思考
时间：{分钟}
```

**Contributor输出模板**：
```markdown
## {Agent名称} - Round 1 想法

1. {想法1}
2. {想法2}
3. {想法3}
```

### Phase 2 - 传递延展

**TeamLead广播**：
```
【脑写法 - Round {N}】传递延展
任务：阅读其他Agent的想法，在此基础上延展或补充新想法
规则：可在他人想法上发展，或提出新想法
```

**Contributor输出模板**：
```markdown
## {Agent名称} - Round {N}

### 延展想法
- 基于{Agent X}的想法{N}：{延展内容}

### 新想法
1. {新想法}
2. {新想法}
```

### Phase 3 - 汇总

**TeamLead汇总所有轮次**：
```markdown
## 脑写法汇总

### 总轮次：{N}
### 总想法数：{N}

| 序号 | 想法 | 原提出者 | 延展者 |
|------|------|----------|--------|
| 1 | ... | Agent A | Agent B, C |
| 2 | ... | Agent B | Agent D |

### 高价值想法（被多次延展）
- ...
```

## 适用场景

- 内向成员较多
- 层级差异较大
- 担心话语权不平等
- 需要高质量想法

## 优势

- 显著降低"谁声音大谁主导"
- 给内向者平等表达机会
- 想法更成熟（多轮延展）

## 注意事项

- 独立阶段严格隔离
- 延展阶段鼓励发展他人想法
- 比口头风暴耗时更长
