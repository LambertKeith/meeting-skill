# 世界咖啡环节推进模板

## 使用说明

本模板用于世界咖啡(World Café)方法的环节推进。通过小组轮换、桌长记录、多轮对话的方式，让不同成员围绕核心问题交叉讨论。

---

## 世界咖啡核心机制

| 元素 | 说明 | Agent适配 |
|------|------|-----------|
| 小组 | 固定小组讨论 | 分组讨论 |
| 桌长 | 记录关键观点 | 每组指定记录者 |
| 轮换 | 小组轮换交流 | 轮换讨论话题 |
| 汇总 | 回收成果 | TeamLead汇总 |

---

## Phase 1 - 设定核心问题

### TeamLead广播消息

```json
{
  "to": "*",
  "summary": "世界咖啡：设定核心问题",
  "message": "【世界咖啡 Phase 1】设定核心问题\n\n任务：确认本轮世界咖啡的核心问题\n时间盒：5分钟\n\n核心问题列表：\n1. {问题1}\n2. {问题2}\n3. {问题3}\n\n分组信息：\n- 轮次：{N}轮\n- 每组人数：{N}人\n- 小组数量：{N}组\n\n输出格式要求：\n## {你的Agent名称} - 问题理解\n\n### 核心问题复述\n- 问题1：{复述}\n- 问题2：{复述}\n\n### 期望讨论方向\n- {方向}\n\n请在{时间}内完成输出。"
}
```

## Phase 2 - Round 1 讨论

### TeamLead广播消息

```json
{
  "to": "*",
  "summary": "世界咖啡 Round 1",
  "message": "【世界咖啡 Round 1】\n\n问题：{问题1}\n\n任务：各组讨论{问题1}，桌长记录关键观点\n时间盒：10分钟\n\n规则：\n- 每组独立讨论\n- 桌长负责记录\n- 不评价观点，鼓励延展\n\n输出格式要求（桌长提交）：\n## 小组{N} - Round 1 记录\n\n### 问题：{问题1}\n\n### 关键观点\n1. {观点}\n\n### 待探讨问题\n- {问题}\n\n### 桌长：{Agent名称}\n\n请在{时间}内完成输出。"
}
```

4. 观点4: {内容}（提出者：{Agent}）

### 待探讨问题
- 问题1: {内容}
- 问题2: {内容}

## 世界咖啡完整流程示例

### 环节推进时间盒建议

| 环节 | 时间盒 | 核心产出 |
|------|--------|----------|
| Phase 1 | 5分钟 | 分组安排、问题确认 |
| Round 1 | 10分钟 | 第一轮观点记录 |
| Round 2 | 10分钟 | 第二轮观点+延展 |
| Round 3 | 10分钟 | 第三轮观点+延展 |
| Phase 4 | 15分钟 | 共识、行动项 |
| **总计** | **50分钟** | **完整会议纪要** |

---

## 注意事项

- **桌长责任**：桌长必须完整记录，不遗漏观点
- **轮换机制**：观点在不同组流动，促进延展
- **收敛慢**：不适合快速决策，适合广泛收集观点
- **最后汇总**：必须形成共识方向，不能只停留在观点列表
